| Metric                | Value          |
|----------------------|----------------|
| Emails Sent          | 20             |
| Emails Opened        | 16 (80%)       |
| Links Clicked        | 12 (60%)       |
| Credentials Submitted| 5 (25%)        |



---

## 🔍 Key Findings
- High open/click rate = realistic phishing simulation.
- Weak awareness about fake login pages.
- Need for regular training and reinforcement.
